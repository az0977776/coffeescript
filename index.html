CoffeeScript

A Brief History

  1. Created by Jeremy Ashkenas in 2009, CoffeeScript was designed as a ``little language that compiles into JavaScript''.
  2. From December of 2009 to December of 2010, the project grew in size, attracting over 300 page hits on the site per day.
  3. The syntax is primarily influenced by Python and Ruby, and the CoffeeScript compiler was initially written in Ruby before being re-written in Coffeescript.

Installation

To Install (with npm):
  npm install -g coffee-script
